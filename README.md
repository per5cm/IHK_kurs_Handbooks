# IHK KURS HANDBOOKS

## English

This repository contains structured handbooks and technical documentation created during the IHK retraining program **“Fachinformatiker für Anwendungsentwicklung”**.

Each section provides a modular Markdown-based guide focused on clarity, reproducibility, and professional lab documentation.

### Repository Structure

| Folder | Description |
|--------|--------------|
| **AD/** | Active Directory setup, user/group management, and password reset guides. |
| **Mail/** | Mail server configuration and setup documentation. |
| **SQL/** | SQL fundamentals, schema design, and database exercises. |
| **VM's/** | Virtual machine installation and configuration (pfSense, Windows, etc.). |

To navigate, open `index_main_menu.md` — it links to all major handbooks.

> All documentation is written in Markdown for version control and compatibility with GitHub. MkDocs oder Docsify to consider for the future.
